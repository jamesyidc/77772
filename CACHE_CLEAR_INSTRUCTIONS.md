## 🔧 清除缓存方法

### 方法1: 硬刷新（推荐）⭐
**Windows/Linux**:
- 按 `Ctrl + Shift + R`
- 或者 `Ctrl + F5`

**Mac**:
- 按 `Cmd + Shift + R`
- 或者 `Cmd + Option + R`

### 方法2: 清除浏览器缓存
1. 打开浏览器设置
2. 找到"清除浏览数据"或"Clear browsing data"
3. 选择"缓存的图片和文件"
4. 点击"清除数据"
5. 刷新页面

### 方法3: 使用无痕/隐私模式
**Chrome/Edge**:
- 按 `Ctrl + Shift + N` (Windows/Linux)
- 按 `Cmd + Shift + N` (Mac)

**Firefox**:
- 按 `Ctrl + Shift + P` (Windows/Linux)
- 按 `Cmd + Shift + P` (Mac)

**Safari**:
- 按 `Cmd + Shift + N`

然后在无痕窗口打开:
```
https://5173-iflt77kp2v3sxzxrncfgd-dfc00ec5.sandbox.novita.ai/signals
```

### 方法4: 开发者工具强制刷新
1. 打开页面
2. 按 `F12` 打开开发者工具
3. **右键点击浏览器的刷新按钮**
4. 选择"清空缓存并硬性重新加载"（Empty Cache and Hard Reload）

## 🐛 如果仍然看到旧版本

### 解决方案：
1. **关闭所有标签页**
2. **完全退出浏览器**
3. **重新打开浏览器**
4. **使用无痕模式**访问页面
5. 如果仍然不行，尝试**不同的浏览器**
